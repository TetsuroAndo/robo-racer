# FTG安全マージンの加速度連動ステート

## 目的
- RPi側FTGの安全マージンをIMU由来の加速度/速度に応じて動的に調整する。
- 反応遅延分は距離側へ反映し、停止距離は速度上限側に反映する。

## 前提
- IMUが無効/未校正の場合は固定マージンへフォールバックする。
- 既存のFTGロジックは維持し、距離の補正のみ追加する。
