# ステア角限界テスト 実装ステート

## 目的
- ステア角の最大動作限界を、安全に確認できるplaygroundロジックを追加する。

## 前提・安全条件
- firmwareは舵角を ±30deg（±3000 cdeg）にクランプする。
- Statusの`steer_cdeg`は適用指令値であり、物理センサのフィードバックではない。
- 30deg超の指令は行わない。
- 異音/音の違和感があれば即座に手動停止できることを必須とする。
