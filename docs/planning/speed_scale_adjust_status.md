# 速度スケール調整ステート

## 目的
- RPiの速度指令とESP32のPWM変換スケールを実測に合わせる。
- 最高速度が極端に低く見える問題を解消する。

## 前提
- RPi側 `SPEED_MM_S_MAX` と ESP32側 `DRIVE_SPEED_MAX_MM_S` は同じ値に揃える。
- 実機での最高速度に合わせて段階的に調整する。
